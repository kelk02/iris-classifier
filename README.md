Iris Decision Tree Classification Project



Project Overview

This project trains a Decision Tree classifier on the Iris dataset.



It performs:

\- Train/Test split

\- Model training

\- Model evaluation

\- Confusion matrix generation



&nbsp;How to Run



&nbsp;1. Activate Virtual Environment



&nbsp;2. Install Dependencies



&nbsp;3. Run the Training Script



Outputs



The script generates:



\- `outputs/confusion\_matrix.png`

\- Printed accuracy score in the terminal
